### Using the Visualizer

1. Click the **"Visualizer"** button or the mini waveform
2. Press **F** key for fullscreen toggle
3. Hover to show/hide controls
4. Press **ESC** to exit

### Customizing Lyrics

1. Click the **âš™ï¸ Settings** button
2. Go to **ğŸ¤ Lyrics** tab
3. Enter lyrics (one line per row)
4. Choose animation style and color
5. Click **Save**

### Creating media.json (Optional)

Create a `media.json` file in your project root:

```json

    "./music/song3.wav"
  ]
}
```
